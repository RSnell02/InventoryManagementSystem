# InventoryManagementSystem
This project allows users to add, remove, or edit items through an easy and clear UI.  
There is a 5% chance of throwing a connection error on load to demonstrate how the UI handles API failurs (with a "Try Again" button)
You can modify current items, add new items, and/or delete items.
Feel free to try out the validation by trying to submit an empty form or test the server validation by trying to create an item with a SKU that already exists.
